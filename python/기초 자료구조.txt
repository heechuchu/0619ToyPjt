컴퓨터는 결국 0과 1만 안다
컴퓨터는 전기 신호로 움직인다
0 = 꺼짐
1 = 켜짐 이 0과 1 하나를 비트라고 함

1비트(bit)는 컴퓨터 데이터의 가장 작은 단위
0 또는 1 이렇게만 표현 가능
ex) 전등 off = 0
     전등 on = 1

8비트 = 1바이트(byte)
01010101 이렇게 8칸 묶인게 1바이트
왜 8비트냐?
1바이트로 표현 가능한 개수 비트는 0 또는 1개이기때문
0~255개 표현가능

문자도 숫자다. 
A = 65
B = 66
이거를 아스키코드라함.
ex ) print(ord('A')) # 65
      print(chr(65)) # A


메모리 크기 차이
bool  → 매우 작음
int   → 보통 4byte
float → 보통 8byte


변수와 메모리 관계

num = 10 이걸 친다
그러면 RAM 메모리 어딘가에 10이 저장됨
그럼 컴퓨터는 메모리에 데이터를 넣고 꺼내며 일함


자료형(Data Type)
컴퓨터는 저장 방식도 구분한다

1. 정수 int 소수점이 없는 숫자
age = 20 
count = 5
정수는 개수,나이,금액,횟수 처럼 딱 떨어지는 값 표현할때만 씀

2. 실수 float 소수점이 있는 숫자
height = 165.5
temperature = 36.7 등 
실수는 키, 몸무게, 온도, 평균, 비율처럼 딱 떨어지지 않는 값을 표현할때씀
average = 87.5
discount_rate = 0.15

정수와 실수 차이
a = 10  이건 정수
b = 10.0 이건 실수 
print(type(a)) # 정수 int
print(type(b)) # 실수 float

주의 할점은 나누기를 하면 정수끼리 계산해도 실수가 될수있다
result = 10 / 2
print(result) 이러면 5.0이 나오고 실수가 되겠지

result = 10 // 2
print(result) 이러면 정수로 받을수있어

정수는 소수점 없는 수.
실수는 소수점이 있는 수.
